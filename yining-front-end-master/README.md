# 前端组

项目启动：

```shell
yarn
yarn dev
```

使用vue复现易宁平台：15天（搭建） + 15天（优化）

技术栈：vue3.0 + ts（规范语法） + AntDesign组件库（表格 + 布局） + yarn（包管理） + AntV（图表） + 百度地图API2.0（地图）+ sass（样式）
